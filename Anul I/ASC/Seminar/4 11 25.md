```asm
bits 32
global start
extern exit, scanf, printf
import exit msvcrt.dll
import scanf msvcrt.dll
import printf msvcrt.dll

segment data use32 class=data:
	n dd 0
	format_c db "%d", 0
	sir times 32 db '0'
            db 'b', 0
	format_a db "%s", 0
segment code use32 class=code:
	start:
	; scanf(format_c, &n)
	push dword n; ca adresă
	push dword format_c
	call [scanf]
	add ESP, 4 * 2
	mox eax, [n]
	cdq
	mov ebx, 2
	mov edi, sir
	add edi, 31
	std

	repeta:
		idiv ebx
		xchg edx, eax
		add al, "0"
		stosb
		xchg edx, eax
		cdq
		cmp eax, 0
		jne repeta

	; printf(format_a, sir)
	push dword sir
	push dword format_a
	call [printf]

	add esp, 4 * 2

	push dword 0
	call [exit]
	add esp, 4
```

# ex 3

```asm
bits 32
global start
extern exit, fopen, fscanf, printf
import exit msvcrt.dll
import fopen msvcrt.dll
import fscanf msvcrt.dll
import printf msvcrt.dll

segment data use32 class=data:
	nf db "a.txt", 0; nume fișier
	ma db "r", 0; mod acces
	desc dd 0
	form db "%d", 0 
	nr dd 0
	err db "Eroare la deschidere fișier", 0
segment code use32 class=code:
	start:

	; fopen(nf, ma)
	push dword ma
	push dword nf
	call [fopen]
	add esp 4 * 2
	cmp eax, 0
	je eroare

	mov [descript], eax
	mov ebx, 0

	repeta:
		; fscanf(desc,form, &nr)
		push dword nr
		push dword form
		push dword desc
		call [fscanf]
		add esp, 4 * 3
		cmp eax, 0
		je gataCitire
		add ebx, [nr]
		jmp repeta

	gataCitire:
		; printf(form, ebx)
		push ebx
		push dword form
		call [printf]
		add esp, 4 * 2
		jmp final

	eroare:
		push dword err
		call [printf]
		add esp, 4 * 1

	final:

	push dword 0
	call [exit]
	add esp, 4
```

## Text Elements
Apeluri de funcții ^OYkKYY5i

Bibliotecă de funcții:
```asm
    extern nume_f1, nume_f2;, ...
    import nume_f1 msvcrt.dll
    import nume_f2 msvcrt.dll
```

Apel:
```asm
    call [nume_f1]
    ; pune pe stivă
    ; -> salt FAR către segmentul de cod care conține definiția funcției apelate
```

Convenții de apel:
    Convenția CDECL:
        - Argumentele se pun pe stivă de la dreapta la stânga
        - Dacă există valoare returnată, atunci aceasta se pune în EAX
        - Regiștrii EAX, ECX, EDX pot fi modificați în interiorul funcției
        - Funcția apelantă eliberează stiva

Afișarea pe ecran
    `printf(format, val1, val2, ...)`
    format = offset către un șir de caractere
    %d = nr întreg în baza 10, dublucuvânt
    %c = un byte, caracter (tot trebuie pus pe un dublucuvânt)
    %u = %d unsigned
    %s = string
    %x = b16

Citire de la tastatură
    `scanf(format, ...)`
    `gets(buffer)` ^vNV5iu7p

1: Scrieți un program care citește de la tastatură un număr întreg și îl afișează în b2 ^cDpEhjcb

2. Scrieți un program care citește de la tastatură un nr întreg și afișează câte cifre are numărul în baza 10 ^MxXV3qxx

Operații cu fișiere text ^Nlo2bEkL

1. Deschiderea unui fișier text
    EAX = `fopen(nume_fisier, mod_acces)`
    Mereu `cmp EAX, 0`
    Dacă EAX este 0 => eroare de deschidere (`je eroare` fix după, unde eroare
afișează pe ecran sau face altceva)

Moduri de acces:
    "r" = read
    "w" = write, scrie de la începutul fișierului
    "a" = append, nu șterge tot ca write
    "r+" = "r" dar fișierul trebuie să existe
    "w+" = "w" dar fișierul nu trebuie să existe
    "a+" = "a" dar fișierul nu trebuie să existe

2. Scrierea în fișier
    `fprintf(descriptor, format, valori)`
    `fwrite(buffer, size, count, descriptor); buffer = adresa la ce vrem să scriem, size = 1`
                                                  `; count = câte caractere vrem să scriem`
    `fscanf(descriptor, format)`
    `fread(buffer, size, count, descriptor)`

    La fread atenție la nr de caract. citite, dacă nu este precizată mărimea atunci mai bine
citește caracter cu caracter și verifică dacă EAX este 0 (unde pune fread numărul de caractere
citite)

3. Închiderea fișierului
    `fclose(descriptor)` ^Uq9JZjyt

3. Se dă un fișier. Fișierul conține nr întregi cu semn separate prin spații. Să se calculeze și afișeze suma numerelor din fișier ^kayqC8bU
